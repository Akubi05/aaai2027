AAAI 2027 anonymous submission source for LatentRisk-VLM.

Main file: latentrisk_vlm.tex
Build: ./compile.bash

The manuscript reports measured results on Waterbirds, CelebA, FairFace, and Flickr30K with frozen Transformers CLIP ViT-L/14 encoders. FACET and SAE are outside the paper scope. The main classification table includes every valid measured result from the selected datasets and distinguishes task-label-only methods from methods that use attribute labels for adaptation or validation selection.

The submission source is anonymous. Author names, affiliations, acknowledgments, and non-anonymous repository links must be added only after acceptance or when preparing a non-anonymous version.
